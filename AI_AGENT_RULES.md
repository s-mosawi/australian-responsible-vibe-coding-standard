# AI Agent Operational Rules

- **Human-in-the-loop required:** AI must not perform irreversible actions without human confirmation.  
- **Restricted commands:** rm -rf, git push --force, chmod 777, terraform destroy, database reset  
- **Code generation:** AI-generated code must follow coding standards and undergo human review.  
- **Data protection:** AI must not access personal information, API keys, or proprietary datasets.  
- **Logging:** AI-assisted actions should be labelled in commits.
