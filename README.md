# Australian Responsible Vibe Coding Standard (ARVCS)

[![Responsible Vibe Coding](https://img.shields.io/badge/Responsible-Vibe%20Coding-green)](https://github.com/responsible-ai-australia/australian-responsible-vibe-coding-standard)

## Overview

This repository provides **practical guardrails and guidance for responsible AI-assisted software development** in Australia.

It includes:

- AI governance and human-in-the-loop rules  
- Privacy and trade secret protection frameworks  
- Trade secret extraction and documentation prompts  
- Automated GitHub compliance checks  
- Structured guidelines for developers and AI coding agents  

This framework aligns with:

- Australian Privacy Principles (Privacy Act 1988)  
- Guidance from the National Artificial Intelligence Centre  
- Best practices for secure and responsible AI development  

---

## Why Use This Standard

- Protects sensitive data and trade secrets  
- Ensures human oversight of AI-generated code  
- Reduces risk of accidental destructive commands  
- Makes your repository **compliant, auditable, and investor-ready**  
- Provides a repeatable process for **innovations and IP protection**  

---

## Getting Started

### Option 1: Use as a Template (Recommended)

1. Click **Use this template** in GitHub  
2. Create a new repository based on this template  
3. All the guardrails and prompts are copied automatically  

### Option 2: Download ZIP

1. Click **Code → Download ZIP**  
2. Extract the files into your repository  
3. Follow the instructions in each `.md` file
