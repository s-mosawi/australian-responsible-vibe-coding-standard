# Australian Responsible Vibe Coding Standard (ARVCS)

This repository follows responsible AI development practices aligned with
Australian guidance on safe AI development.

These guidelines apply to developers, contributors, and AI coding assistants
interacting with this codebase.

## Core Principles

- **Accountability:** AI-generated code must be reviewed by a human before deployment.  
- **Human Oversight:** AI agents must not perform critical operations without human approval.  
- **Privacy Protection:** AI tools must not receive personal data (names, financial info, health data).  
- **Intellectual Property Protection:** Do not include trade secrets or proprietary data in AI prompts.  
- **Security:** AI-generated code must undergo security review before production.
